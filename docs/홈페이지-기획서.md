# 룸메이커 홈페이지 기획서

**작성일:** 2026-07-04
**상태:** 기획 중
**목표 오픈:** 2026년 8월 (책 출판 전후)

---

## 1. 목적

| 목적 | 설명 |
|------|------|
| B2B 신뢰도 | 의뢰 담당자가 검색했을 때 "전문가"로 보이기 |
| B2C 판매 연결 | 인스타 외 경로로 온라인 방탈출 구매 유입 |
| 브랜드 허브 | 책 / 인스타 / 스마트스토어를 한 곳에서 연결 |

---

## 2. 타깃

| 타깃 | 유입 경로 | 원하는 행동 |
|------|----------|------------|
| B2C — 퀴즈 좋아하는 개인 | 인스타, 책, 검색 | 온라인 방탈출 구매 |
| B2B — 행사/기업 담당자 | 검색, 명함 QR | 의뢰 폼 작성 |

---

## 3. 페이지 구조

```
홈 (단일 페이지 스크롤 구조)
│
├── 히어로 섹션
│   ├── 브랜드명 + 한 줄 소개
│   ├── 인스타 팔로워 수 (사회적 증거)
│   └── CTA 버튼 2개: [방탈출 구경하기] [행사 의뢰하기]
│
├── 온라인 방탈출 섹션
│   ├── 상품 카드 목록 (썸네일 / 난이도 / 인원 / 가격)
│   └── [구매하기] → 스마트스토어 연결
│
├── 책 소개 섹션
│   ├── 표지 이미지
│   ├── 한빛라이프 출판 소개 문구
│   └── [구매하기] → 출판사/서점 링크 (8월 이후 활성화)
│
├── B2B 서비스 섹션
│   ├── 야외 방탈출 / 기업 워크숍 / 행사 기획 소개
│   ├── 레퍼런스 (안동 페스티벌 이후 추가)
│   └── [행사 의뢰하기] → 의뢰 폼
│
├── 의뢰 폼
│   ├── 이름
│   ├── 행사 종류 (야외 방탈출 / 기업 워크숍 / 기타)
│   ├── 날짜 / 인원 / 예산 범위
│   └── 연락처 (이메일 or 연락처)
│
└── 푸터
    ├── 인스타 링크
    └── 이메일
```

## 4. 기능 우선순위 (MoSCoW)

### Must Have (Phase 1 — 오픈 시)
- [ ] 온라인 방탈출 상품 목록 + 스마트스토어 연결
- [ ] B2B 의뢰 폼
- [ ] 책 소개 섹션 (출판 예정 배너)
- [ ] 인스타 링크 + 팔로워 수

### Should Have (Phase 2 — 판매량 붙으면)
- [ ] 샘플 방탈출 체험 (첫 문제 무료)
- [ ] 구매자 후기 섹션
- [ ] 레퍼런스 페이지 (안동 페스티벌 등)

### Could Have (Phase 3 — 게임 서버화 이후)
- [ ] 홈페이지 자체 결제
- [ ] 구독 모델
- [ ] 회원 시스템

### Won't Have (지금은)
- [ ] 앱 (판매량 데이터 쌓인 후 재검토)
- [ ] 블로그/콘텐츠 섹션 (인스타가 대신함)
